Bonding Curve Mechanics

The bonding curve is a mathematical function that determines token price based on circulating supply.

Price Formula

  Price = f(Supply)

  Where the curve follows:
  
  P(s) = k √ó s¬≤
  
  k = constant determined at launch
  s = current circulating supply
  
  As more tokens are bought:
  - Supply (s) increases
  - Price P(s) increases quadratically
  - Each subsequent token costs more

Key Properties

Property Description Benefit
Deterministic Pricing Price is calculated mathematically, not by order book No price manipulation, transparent pricing
Instant Liquidity Always able to buy or sell against the curve No waiting for counterparty, no slippage on small orders
Fair Launch No presale, everyone buys from same curve Equal opportunity, no insider advantage
Price Discovery Market determines value through buying/selling Organic price based on actual demand

Graduation Process

When a token reaches the graduation threshold, it migrates from pump.fun to Raydium.

Graduation Threshold

~$69,000 Market Cap

When a token's market capitalization reaches approximately $69,000, it automatically triggers the graduation process and migrates to Raydium DEX.

What Happens During Graduation

  1. Bonding Curve Closes: No more buying/selling on pump.fun
  2. Liquidity Calculation: Total SOL in curve is calculated
  3. Pool Creation: Raydium AMM pool is created
  4. Liquidity Migration: SOL and tokens are added to the pool
  5. Trading Resumes: Token is now tradeable on Raydium

Deflationary Mechanisms

ClawPad tokens can become deflationary through buyback and burn programs:

  • Creator-Funded Burns: Creators use their 1% fee to buy back and burn tokens
  • Community Burns: Community members can voluntarily burn tokens
  • Scheduled Burns: Some tokens implement scheduled burn events
