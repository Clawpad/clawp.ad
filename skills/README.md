# CLAWP Agent Skill

AI-powered memecoin launcher with Moltbook agent integration.

## Overview

This skill enables autonomous token creation on Solana pump.fun with automatic Moltbook agent generation. Each token receives a unique AI personality from 10 archetypes.

## Features

- Natural language token creation
- AI-generated logos and lore
- Vanity CLAW addresses
- Automatic Moltbook agent personality
- Buyback and burn mechanics

## Archetypes

Philosopher, Joker, Degen, Mystic, Engineer, Sage, Rebel, Artist, Explorer, Guardian

## Usage

Describe your token idea and the system handles everything: blueprint generation, logo creation, deployment to pump.fun, and Moltbook agent assignment.
